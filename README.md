# Todo Dashboard

A personal dashboard web app to efficiently manage daily tasks with a clean interface, light/dark mode, and reminders.

---

## Features
- ✅ Add, edit, and delete tasks
- ⏰ Schedule tasks with date & time
- 🔔 Set reminders before task time
- 📂 Categorize tasks: Current, Upcoming, Completed
- 🔍 Search and filter tasks
- 🗑 Bulk select and delete tasks
- 🌙 Light and Dark theme toggle
- 🌐 Multi-language support (English & Arabic)
- 💾 Persist data using LocalStorage

---

## Technologies Used
- HTML, CSS, JavaScript
- React
- Vite (with HMR for fast development)
- ESLint rules applied for cleaner code
- Browser Notifications API
- LocalStorage for persistence
- **Development assisted with AI tools**: Trae AI editor using **Gemini-3-Flash-Preview** for code suggestions and problem-solving
